## 2024-05-24 - React.memo for Search Input Re-renders
**Learning:** The frontend application exhibits an architectural pattern where search inputs in root components (like `Home.jsx` and `Explore.jsx`) trigger full page re-renders on every keystroke. This causes performance issues when rendering large lists.
**Action:** List item components used within these pages (e.g., `ModelCard`) should be wrapped in `React.memo` to prevent O(N) re-rendering performance bottlenecks.
